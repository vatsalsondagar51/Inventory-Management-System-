📦 Inventory Manager – Web-Based Inventory Management System

Inventory Manager is a modern web-based inventory management system designed to help businesses efficiently manage products, suppliers, orders, and deliveries in a centralized dashboard.

The application provides a simple ERP-style workflow that allows users to track stock levels, manage supplier contacts, create orders, and monitor delivery status.

🚀 Features
🔐 Authentication

User Sign Up

User Login

Secure dashboard access after authentication

📊 Dashboard

The dashboard provides an overview of inventory activity including:

Total products in inventory

Low stock alerts

Pending orders

Deliveries currently in transit

Recent activity tracking

This helps users quickly understand the current state of inventory operations.

📦 Inventory Management

The inventory module allows users to manage products and stock levels.

Features include:

Add new products

Edit product information

Delete products

Track stock quantities

Set product SKU and category

Define low stock threshold

Search products quickly

Product details include:

Product name

SKU

Category

Quantity

Unit of measure

Stock status

🧾 Orders Management

The orders module allows users to create and track purchase or sales orders.

Features include:

Create new orders

Select customer or supplier

Add multiple items to an order

View order status

Edit orders

Order details include:

Order ID

Customer / Supplier

Items included

Date

Order status

🚚 Delivery Tracking

The delivery module tracks shipments associated with orders.

Features include:

Create delivery record

Assign carrier (DHL, FedEx, UPS, Blue Dart)

Enter tracking number

Update delivery status

Track estimated delivery date

Delivery status types include:

Pending

Picked Up

In Transit

Out for Delivery

Delivered

🏭 Supplier Management

The supplier module manages supplier contact information.

Features include:

Add supplier

Edit supplier details

Delete supplier

Store contact information

Supplier details include:

Company name

Contact person

Phone number

Email

Address
